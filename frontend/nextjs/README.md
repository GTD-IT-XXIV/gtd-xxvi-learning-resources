Bagian ini bisa learn as you go.

## Goals

1. Ngerti perbedaan Server-Side Rendering (SSR) dan Client-Side Rendering (CSR)
2. Ngerti kapan harus pake CSR di Next.js.
3. Ngerti cara routing (App Router) di Next.js.
